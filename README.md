# 🎓 EduMentor AI

> **Learn Smarter. Practice Better. Achieve More.**

EduMentor AI is an AI-powered learning platform that transforms study materials into interactive learning experiences. It helps students understand concepts faster, revise efficiently, and prepare confidently through intelligent AI-assisted learning tools.

---

# 🚀 Overview

EduMentor AI is designed to simplify the learning process by combining Artificial Intelligence with modern educational tools. Students can upload study materials and instantly generate explanations, summaries, quizzes, flashcards, viva questions, and AI-powered conversations based on their uploaded documents.

The platform provides an organized learning workspace that helps students manage subjects, units, documents, study planning, and learning progress from one centralized application.

# ⚙ Installation

## Clone Repository

```bash
git clone https://github.com/YOUR_USERNAME/EduMentor-AI.git
```

```bash
cd EduMentor-AI
```

---

## Create Virtual Environment

Windows

```bash
python -m venv venv
```

Activate

```bash
venv\Scripts\activate
```

---

## Install Dependencies

```bash
pip install -r requirements.txt
```

---

## Configure Environment Variables

Create a `.env` file in the project root.

Example:

```env
GROQ_API_KEY=your_groq_api_key
GEMINI_API_KEY=your_gemini_api_key
```

---

## Run Application

```bash
python -m streamlit run run.py
```
